Java学习P1-05 注释

注释的作用

  • 被注释的内容，编译器会忽略
  • 主要作用：对代码进行解释说明。
  • 原则：适当的注释。自解释的代码
    • 终极目标：不写注释，别人也能看懂你的逻辑
  • 副作用：暂时停用部分代码

三种注释

  • 1、单行注释：// 本行中//之后的为主
  • 2、多行注释：/* */
  • 3、JavaDoc
