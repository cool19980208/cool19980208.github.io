P3-4 类的基本语法

类的基本语法

类的基础知识

  • 1、private：只能类内部访问；public：其他类也可以访问
  • 2、类的定义由成员变量和方法组成
    • 方法：动作
      • 大多数情况都是public
    • 成员变量：状态
      • 大多数情况都是private

Java中定义一个类的语法

创建类的对象

  • 可以声明Person类型的变量：Person p1；

方法调用

类的另一种解读

  • 类是把对象的若干个状态(成员变量)和相关的方法聚集到一个模板中的机制
  • new一个类就是根据类的模板实例化出来一个对象
    • 不同的对象之间互不干扰

static

  • static方法和对象无关，因此可以直接通过类名调用
    • 直接用类目. 去访问static方法
    • 用p1.hi()的话，是没有意义的，因为static方法与对象无关
    • static方法是不依赖于类的成员变量的
    • 好比:String也有static方法
  • 非static方法需要通过对象调用
    • 非static方法是依赖于成员变量的，所以不能直接使用类目去调用
