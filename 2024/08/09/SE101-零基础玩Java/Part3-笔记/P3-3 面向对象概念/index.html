P3-3 面向对象概念

面向对象概念

说明
1、for、while、变量、方法、数组等这些都是“面向过程”的语法，Java还提供了“面向对象”的语法。“面向对象”不是用来替代“面向过程”的，而是一个升华和补充。
2、初学者不可能完全搞明白“面向对象”，搞明白语法即可

类和对象

对象发送消息

  • 消息：不同的对象之间可以互相发送“消息”
    • 好比下面这张图！
    • 关羽是对象(人类的实例)；马也是对象（动物类中马的实例）
    • 比如关羽可以向马 发出消息：驾、驾、驾 ；冲….

封装

  • 马自身把步伐、呼吸、进食等“封装”起来。其他的对象可以向马发送消息
    • 比如发送：喝水，追等

属性

  • 不同的“马”类对应的实例属性都不相同

继承

  • 继承是用来简化类型声明的
