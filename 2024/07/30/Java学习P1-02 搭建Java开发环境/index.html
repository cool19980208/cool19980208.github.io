Java学习P1-02 搭建Java开发环境

为什么需要搭建开发环境

快速搭建步骤

JDK+IDEA

  • JDK：编译、运行Java程序，基本类库
  • IDEA：集成开发环境（IDE）
    • 还有Eclipse、NetBeans等
  • 版权意识：破解对于版权来说不好

JDK的版本

  • 1.4……8、9、10、16等
  • 为什么主流仍然是8/1.8
    • 因为大多数公司都是8，为了稳定，不敢动而已
    • 但是学技术的还是要会新的东西的
  • 学的东西用高版本也兼容

JDK安装

  • 浏览器搜索“JDK8下载”，官方名“Java SE Development Kit 8”
  • Windows、MAC都一样
    • 官方下载放心
    • 国内第三方网站的下载慎点，太多坑了
      • 都是捆绑软件
    • 初学者不用配置环境变量
  • JDK8的下载地址
    • https://www.oracle.com/java/technologies/downloads/#java8-windows
    • 直接傻瓜式安装即可，修改一下“安装路径”就行。看图一步步来

IDEA安装

  • 搜“IDEA Java” 或官网www.jetbrains.com.cn
  • 直接傻瓜式安装即可，修改一下“安装路径”就行。看图一步步来
